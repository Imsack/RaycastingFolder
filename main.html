3D rendering using Raycasting and Raymarching

Main Raycasting Raymarching

Raycasting and Raymarching are 3D rendering techniques that include casting rays (represented as lines or vectors) in a virtual scene and letting the rays intersect with objects in the scene to gather information about how to render them onto the screen. The biggest difference between Raycasting and Raymarching in this instance is that we will be using raycasting to generate psuedo-3D scenes while raymarching will be used for real 3D rendering.
